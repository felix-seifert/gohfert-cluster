# Certificates for TLS

The certificate for trusting the HTTP traffic to private website deployed behind Traefik of Kubernets can be found on
the unenencrypted page [http://ca.gohfert.com](http://ca.gohfert.com).

We have an [explanation](../explanations/ca-certs.md) why this certificate is not available publicly, resulting in web
browsers not being able to retrieve it automatically.
